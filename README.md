# 🧞‍♂️ PicGenie - AI Text to Image Generator

PicGenie is a full-stack AI project that generates images from text prompts using advanced generative AI models. It's built with the **MERN** stack and integrates with **Replicate API** to transform your imagination into visuals.

---

## ✨ Features

- 🎨 Generate images from any text prompt using AI (e.g., “a futuristic city in the clouds”)
- 🔁 Regenerate images with the same or new prompt
- 💾 Download the generated image
- ⚡ Fast response via server-side processing
- 🧪 Clean and responsive UI with Vite + Tailwind CSS

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios

### Backend
- NodeJs & ExpressJs
- MongoDB with Mongoose
- JWT for authentication
- Razorpay for payment integration
- Clipdrop API for image generation
